【ウェーブ数の設定】
・値の加算 : →キー, Dキー
・値の減算 : ←キー, Aキー

【1ウェーブごとに出現する的数の設定】
・ウェーブ数の設定に戻る : F1キー

・値の加算 : →キー, Dキー
・値の減算 : ←キー, Aキー

・ウェーブ数の加算 : ↓キー, Sキー
・ウェーブ数の加算 : ↑キー, Wキー

【ステージデータの設定】

[的のタイプ | 出現時間 | 出現してから消えるまでの時間の設定]
・値を加算する : Wキー, 右矢印キー
・値を減算する : Sキー,  左矢印キー

[的の座標設定]
・座標の移動 : マウスを移動させる
・座標の設定 : 左クリック

[設定する的の切り替え]
・値の加算 : →キー, Dキー
・値の減算 : ←キー, Aキー

[設定するウェーブ数の切り替え]
・ウェーブ数の加算  : Dキー + SHIFTキー, ↓キー + SHIFTキー
・ウェーブ数の減算  : Aキー + SHIFTキー, ↑キー + SHIFTキー

【システム関係】

・ウェーブ数とターゲット数の再設定　	　　　 :　F1キー
・ステージをロードする		　　　 :　F4キー
・ステージをセーブする		　　　 :　F5キー
・設定するシーンの移動, 設定する的の切り替え　:　SPACEキー
